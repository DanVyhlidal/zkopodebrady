# ZKO Poděbrady Website

This is the repository for the [ZKO Poděbrady](https://zkopodebrady.cz) website.
It is a modern, performant static web application built to replace a legacy WordPress setup, offering improved speed, easier maintainability, and a streamlined content management experience.

## Technologies Used

- **React 19** & **TypeScript**
- **Vite** as the build tool and development server
- **React Router DOM** for client-side routing
- **Decap CMS** for a Git-based, Markdown-driven content management workflow
- **React Markdown** & **Front Matter** for rendering CMS content

## Key Features

- **Aktuality (News):** A dynamic news feed managed completely via Decap CMS using Markdown files.
- **Galerie (Gallery):** A streamlined photo gallery that dynamically fetches albums and photos from Google Drive. It features lazy loading and a click-to-zoom lightbox for an optimal viewing experience.
- **Členové (Members):** A card-based directory of club members and their dogs.
- **Static Pages (Výcvik, Kontakt):** Fast-loading informational pages including a localized Google Maps embed for easy navigation to the training grounds.
